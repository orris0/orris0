A COMPUTER FOR £15?

The UK not for profit organization Raspberry Pi is hoping to sell
a computer for £15 (or less if the take up is good).
It is no bigger than a USB connector, it has an ARM chipset,
a wi-fi chip, a version of the BASIC progrmming language
and it is preloaded with the Linux operating system. The primary purpose
for this innovation is as a teaching tool for UK schools where children
are bored stiff with the current curriculum. You can read more about it at
http://www.wired.co.uk/news/archive//2011-01/20/raspberry-pi-computer
